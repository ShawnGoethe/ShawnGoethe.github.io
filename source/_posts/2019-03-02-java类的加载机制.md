# what is the loading of class

类加载即：

将编译class文件中的二进制数据读到内存中方法区，然后在堆区通过java.lang.Class实例化对象，对方法区的数据进行操作

![img](http://www.itmind.net/assets/images/2017/jvm/jvm-1.png)

该加载过程包含首次使用加载，以及预加载

## 加载class文件的方式

- 本地
- 网络
- zip，jar文件中
- 数据库
- 动态编译
